Getting Started

Welcome to the CRM System documentation. This guide will help you get started with our platform.

Installation

To install the CRM System, follow these steps:

  1. Clone the repository
  2. Install dependencies with pip install -r requirements.txt
  3. Configure your database settings
  4. Run the application with flask run
